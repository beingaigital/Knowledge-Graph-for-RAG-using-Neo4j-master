Section: 2.2 华与华品牌三角形
Section: 华与华品牌三角形，就是产品结构、话语体系和符号系统。
Section: 产品结构：不管你要做一个什么样的品牌，首先得有产品，产品是品牌的本源。不管你是物
Section: 理意义上的产品还是服务产品，甚至观念上的产品，总之你一定是有产品才会去建立品牌。
Section: 话语体系：有产品就一定会有产品命名、产品定义，这个命名和定义就是品牌的话语体系，
Section: 是品牌的文本传达。
Section: 华与华还将企业的事业理论、产品科学、品牌话语、企业文化等，全都纳入品牌话语体系，
Section: 并入品牌管理中。
Section: 符号系统：每个产品或品牌，它都有一个感官上的体验。看着像什么样子，听上去是什么声
Section: 音，摸起来是什么触感，闻起来是什么气味，吃进去是什么味道……这些品牌的感官信号就
Content: 是它的符号系统。
Section: 前面我们说营销的唯一理论是4P，当我们要改变一个营销策略时，我们有4张牌可以打：要
Content: 么调整产品，要么调整价格，要么调整渠道，要么调整推广。
Section: 那我们回过头来看品牌，如果我们要解决某个品牌的问题，要么就解决产品问题，比如产品
Content: 的开发、产品推出的次序；要么就调整话语体系，调整我们对外宣传的内容，比如产品文
Section: 案、广告话语等；要么就重新设计符号系统，比如包装设计、品牌角色等。
Content: 我们也可以从自然辩证法的角度来理解：产品终归要体现为一个东西，是物质；而话语体系
Content: 是用来解释产品的，是意识；物质决定意识，意识对物质有能动作用。华与华方法说，一切
Section: 营销传播都是符号的编码和解码，超级符号运用的是人类的集体潜意识。
Content: 所以产品结构、话语体系和符号系统这三条边，实际上也是物质、意识和潜意识这三个方
Content: 面。

Content: 化契约。
Content: 什么叫共同文化契约？这是符号学的一个基础。比如，说出“狗”这个词，我们所有人都知
Content: 道那是指“狗”，因为这是我们所有人都约定了那个东西就叫“狗”。

Content: 似于这样的约定，我们就把它称为文化契约。

Content: 华与华方法认为：命名就是成本，命名就是召唤，命名就是投资。
Section: 1）命名就是成本：命名的第一原则是成本，品牌要尽可能选择那些成本低的名字。
Section: 成本低就是要降低理解成本、识别成本、记忆成本、传播成本。比如，电脑品类中，“苹
Content: 果”的命名就要好过“戴尔”，因为苹果是全世界最普通、最常见的一种水果，人人都熟
Content: 悉，看见一次就记住了。

Section: 文名KBOY&KGIRL，绝大多数家长都念不出来，更别说理解、记忆和传播了。
Section: 2）命名就是召唤：命名的第二原则是召唤，你的品牌名，是否有指向性很强的召唤，是否
Content: 召唤了价值，是否下达了明确的指令。
Section: 比如，丰田曾经将其在中国的品牌和产品进行了全线改名，如将“陆地巡洋舰”改为“兰德
Content: 酷路泽”等。这便是将召唤性超强而成本超低的好名字改为无任何召唤性的名字。
Section: 召唤，是词语的能动性，它能将品牌的价值和顾客召唤来，如“陆地巡洋舰”能将其价值和
Content: 顾客召唤来，“兰德酷路泽”就不能。
Content: 在实际的市场调研中，很多店员和家长都反映，“今童王”的“童”字听起来有些幼稚，很
Section: 多十几岁的大孩子一看到“今童王”三个字，就觉得是小孩子穿的童装品牌，心理上较为抵
Content: 触，即使家长把衣服买回家也不会穿。

Content: 少年群体。

Content: 最大效益。
Section: 20多年来，今童王曾频繁改动中英文名，每一次改动都是品牌资产的重大流失。
Content: 从成本、召唤和投资的角度来看，“今童王”确实不是好名字，但已在渠道大规模沉淀，合

Content: 消费者的购买旅程：消费者的四个角色
Section: 任何产品的消费旅程，消费者都存在四个角色：受众—购买者—体验者—传播者。
Section: 第一阶段是受众阶段：这个阶段的消费者是茫然的、容易遗忘的，品牌要做的是从茫然中唤
Content: 醒消费者、刺激消费者；
Section: 第二阶段是购买者阶段：这个阶段的消费者从繁杂的购买环境中搜寻信息，做出决策，品牌
Content: 要做的是提供购买理由、购买指南，下达购买指令，打动消费者购买，促成销售；
Section: 第三阶段是体验者阶段：这个阶段的消费者怀着对产品、品牌的期待和验证心理，品牌要提
Content: 供的是体验的惊喜点，创造符号化，形成仪式化；
Section: 第四阶段是传播者阶段：每一位消费者都是最好的传播者，也就是口碑，这个阶段的消费者
Section: 是感性的、无意识的，品牌要做的是设计一句话让消费者主动替品牌传播。

Content: 消费者有四个角色，就是消费的四个阶段，四个情景语境，四个不同的时间、地点、信息、
Content: 反应、目的、方式、内容等。
Content: 这四个情景语境是：购买前、购买中、使用中、使用后。
Content: 购买前：消费者还没有到购买场所，没有到卖场，没有上购物网站。商品还在商店或仓库，
Content: 消费者还没有买它，但是消费者从广告上、从亲友口中得到了它的信息；或者在朋友家看到
Content: 了它，先留个心；或者直接看到了这个商品。记住这三个情景：看到或听到广告，听到别人

Content: 购买中：消费者到达了购买场所，到了卖场，或上了购物网站，正在溜达、观察、体验。
